## 介绍

avue-cli是一款基于avue和element-ui完全开源、免费的企业后端产品前端集成解决方案，采用最新的前端技术栈，已经准备好了大部分的项目准备工作，你可以快速进行二次开发

## 开发

```
# 克隆项目
git clone https://gitee.com/smallweigit/avue-cli.git

# 进入项目
cd avue-cli

# 安装依赖
npm install --registry=https://registry.npm.taobao.org

# 启动服务
npm run serve

```
## 功能
- 全局错误日志记录
- vuex持久化存储
- 主题色切换
- 消息提醒
- 锁屏
- 数据展示
- 登录/注销
  - 用户名登录
  - 验证码登录
  - 第三方登陆(QQ,微信)
- 权限验证
- 第三方网站嵌套
- CRUD(增删改查)
- FORM(动态生成)
- 阿里巴巴图标库(在线调用)
- tag标签操作
- 环境变量
- 表格树
- 引导页
- 数据持久化
- 剪切板
- 灰度化
- 系统管理
  - 用户管理
  - 角色管理
  - 菜单管理
- 高级路由
  - 动态路由
  - 参数路由
- 更多功能开在开发
